Commandes:
-git init
Initialise le projet git dans le dossier sélectionné. Si on est dans le dossier X , et que l'on tape en ligne de commande (LDC):git init, on indiquera donc que l'ont va commencer à travailler dans le dossier X.
-git status
Permet de consulter les modifications en attente qui n'ont pas été encore "upgraded".
-git add X
permet d'ajouter les modifications en attentes, du fichier X.
-git commit -m "message"
permet d'enregistrer le nom de l'ajout sur Github.
-git log
Permet d'avoir le log de tous les commits (ajouts) qui ont été effectués.log
-git log --oneline
Version minimaliste de git log.
-git push -u origin master
Permet d'effectuer un "push" vers le site Github.com. Cad que l'on "pousse" notre fichier mis à jour sur git, vers la plateforme communautaire Github.
-ssh -T git@github.com
Configure pour la clé usb oclock, la fenêtre de LDC mint pour qu'elle puisse intéragir avec GitHub sans avoir à rentrer le login/MdpCommandes:
